
# MERN Authentication System - Backend

## 📌 Features
- User authentication using JWT  
- Secure cookie-based token storage  
- Email verification using OTP  
- Password reset with OTP  
- Email sending using Nodemailer and SMTP  
- Cross-origin requests handled using CORS  
- Cookie parsing using cookie-parser  

## 🛠️ Tech Stack
- Node.js  
- Express.js  
- MongoDB  
- JWT Authentication  
- cookie-parser  
- Nodemailer (SMTP)  
- CORS
